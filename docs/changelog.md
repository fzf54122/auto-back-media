# 更新日志

本文件记录了自动备份媒体系统的版本历史和主要功能变更。

## [1.0.0] - 2023-12-13

### 新功能

- 系统架构设计与基础框架搭建
- 用户认证与授权系统
  - 用户登录、退出功能
  - Token生成与刷新机制
  - 权限验证中间件

- 用户管理模块
  - 用户CRUD操作
  - 用户密码修改
  - 用户角色关联

- 基础模块
  - 角色管理：角色CRUD和权限分配
  - 部门管理：部门层级结构管理
  - 菜单管理：系统菜单配置
  - 文件管理：文件上传功能
  - 审计日志：系统操作日志记录
  - API权限管理：接口权限控制

- 管理模块
  - 媒体用户管理：媒体平台账号管理
  - 任务管理：系统任务调度
  - 网站管理：外部网站集成配置

- 系统模块
  - 系统设置：全局配置管理

### 技术栈

- 后端框架：FastAPI
- 数据库：PostgreSQL
- ORM：Tortoise-ORM
- 认证：JWT
- 文档：MkDocs + Material
- 开发工具：Black、isort、Pylint、Pytest、Coverage、Ruff等

## [0.1.0] - 2023-11-29
